# dotfiles

## Setup

- Linux

```
./init.sh
# or
./init.sh -f
# to force update
```

- Windows

```
./init.ps1
# or
./init.ps1 -f
# to force update
```

## dependencies and so on

### powershell

- `starship`

### shell(zsh)

- `starship` (and `lsb-release` command to increase performance)
- `exa`
- `zoxide`
